## Your Mission

Explore this entire codebase in parallel and produce `.sisyphus/context.md` — a living reference document that any agent can read to immediately understand the project without re-exploring.

## Exploration Steps (Run in Parallel)

Launch these simultaneously:

**Wave 1 (parallel):**
1. List all top-level directories and files — understand project structure
2. Read package.json / go.mod / Cargo.toml / requirements.txt — understand tech stack and dependencies
3. Find entry points (main.ts, index.ts, app.tsx, main.go, etc.) — understand where execution starts
4. Find README.md — understand stated purpose and setup instructions

**Wave 2 (parallel, based on Wave 1 findings):**
5. Read the 3–5 most important source files identified in Wave 1
6. Find test files — understand testing patterns and coverage approach
7. Find configuration files (.env.example, config/, settings) — understand configuration approach
8. Grep for common patterns (error handling, logging, auth, database access)
